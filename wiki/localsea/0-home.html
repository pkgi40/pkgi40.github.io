Home

This tutorial collection guides users to understand Papyrus4Manufacturing (P4M) and use it from simple to complex use cases. All of the use cases are implemented and tested in LocalSEA, the Industry 4.0 & 5.0 testbed under the management of DRT/LIST/DILS.

Walkthroughs

Note: These following tutorials address users who have at least some basic understanding about AAS and P4M. If you have no idea about them, please check other documents and tutorials at the P4M’s Documentation for more information.

  • 01: Led blink

Monitor and Control a led to blink using an AAS DT of led. This tutorial allows understanding the mechanism behind reading an AAS DT’s property.

  • 02: T3WP moving

Control a the TurtleBot3 Waffle Pi robot to move forward/backward. This tutorial allows understanding the mechanism behind invoking an AAS DT’s operation.

  • 03: UR3e controlling

Control a UR3e robot for basic Pick and Place tasks. This tutorial allows understanding the mechanism behind writing a value on an AAS DT’s property.

  • 04: Multiple protocols

Use +10 network protocols to change color of an object. This tutorial guides using different network protocols for connecting an AAS DT and its asset.

  • 05: AAS UML & AAS JSON conversion

Convert an AAS information model between UML and JSON. This tutorial guides converting from AAS JSON to AAS UML format and vice-versa.

  • 06: AAS & OPC UA Information Models conversion

Convert an AAS information model into OPC UA information model. This tutorial guides converting from AAS information model into OPC UA information model.

  • 07: Eclipse Dataspace Components (EDC) connector

Use Eclipse Dataspace Components’s connector to connect a Dataspace. This tutorial guides using EDC connector in practice.

  • 08: GUI with Chatbot

Use a Graphic User Interface (GUI) that supports a GPT-like chat bot. This tutorial guides using the GUI and the Chatbot to interact with an AAS DT.

  • 09: Security

Applying different security modes for AAS digital twins. This tutorial guides using different security modes of an AAS DT.

LocalSEA

LocalSEA is a small-scale and didatic testbed under the management of DRT/LIST/DILS. The tests primarily focus on the software components and technologies developed by the department. It is equipped with compact industrial equipment, IoT devices, and robots.
