Update Sites

Setup Eclipse Papyrus with Update Sites

  • The most basic procedure for installing Eclipse Papyrus consists in installing the Eclipse Modeling Package for your own platform.
  • Then, you have to use the discovery interface ("Help" > "Install New Software" > "Modeling" ) and select Papyrus for UML.
  • You may install the latest release of Eclipse Papyrus by following the instructions below "Update sites".

Update Sites

Eclipse provides facilities for adding new software to the platform or updating software in the system. In all cases, the site location (i.e. the Web URL or the archived Update Site provided above) is the only required item to update or install a software within Eclipse.
